Introducción

La regresión lineal es una técnica de análisis de datos que predice el valor de una variable desconocida (dependiente) utilizando el valor de otra variable relacionada y conocida (independiente), modelando esta relación mediante una ecuación lineal. Como explica el portal AWS: "La regresión lineal es una técnica de análisis de datos que predice el valor de datos desconocidos mediante el uso de otro valor de datos relacionado y conocido" (Amazon Web Services, s.f.). De forma similar, IBM define que "La regresión lineal es una técnica de análisis utilizada para predecir el valor de una variable dependiente basada en el valor de una o más variables independientes" (IBM s.f). En este trabajo aplicaremos la regresión lineal para predecir el precio de frutas según la cantidad producida y el costo de transporte, mostrando cómo este modelo puede ser utilizado en un contexto agrícola y económico.

1. Regresión Lineal

En el contexto del Machine Learning, los algoritmos de regresión lineal se entrenan con datos históricos (etiquetados) para calcular la ecuación que mejor ajusta la relación entre variables, y luego predecir valores desconocidos. Por ejemplo, AWS menciona que en machine learning, "los programas de computación denominados algoritmos analizan grandes conjuntos de datos ... para calcular la ecuación de regresión lineal" (Amazon Web Services, s.f.). FlowHunt destaca que en inteligencia artificial y aprendizaje automático, la regresión lineal "suele ser el modelo introductorio debido a su simplicidad y efectividad para tratar relaciones lineales" (FlowHunt., s.f.). 2. Linealidad

En Machine Learning, la linealidad se refiere al concepto de que la relación existente entre las variables de entrada (predictoras) y la variable de salida (respuesta) puede ser representada mediante una línea recta o un plano. En términos sencillos: cuando se altera una variable, el impacto en la salida es siempre constante y proporcional. Estas propiedades se conocen como el principio de superposición. En esta definición, x no es necesariamente un número real, pero en general puede ser un elemento de cualquier espacio vectorial. Una definición más especial de función lineal, que no coincide con la definición de mapa lineal, se usa en matemáticas elementales.

3. Independencia

En Machine Learning, la linealidad se refiere al concepto de que la relación existente entre las variables de entrada (predictoras) y la variable de salida (respuesta) puede ser representada mediante una línea recta o un plano. En términos sencillos: cuando se altera una variable, el impacto en la salida es siempre constante y proporcional. La independencia en el aprendizaje automático implica que las variables empleadas para entrenar el modelo no deben tener una dependencia mutua directa. En otras palabras, cada una debe contribuir con información diferente y no estar "repitiendo" lo mismo. Desde una perspectiva estadística, la independencia se refiere a que las características del conjunto de datos no muestran multicolinealidad y que los errores o residuos del modelo no están correlacionados. Esto garantiza que el modelo pueda aprender patrones auténticos y no ruido redundante.

4. Homocedasticidad

En el contexto de machine learning la homocedasticidad se refiere a que los errores (residuos) del modelo tienen una varianza constante para todos los valores de las variables independientes. Esto quiere decir: Si graficamos los residuos vs. los valores predichos, deberían estar más o menos dispersos de manera uniforme (como una nube), Y no deberían verse patrones como un "abanico" que se abre o se cierra. Un ejemplo entendible puede ser que: Supongamos que predecimos los salarios en función de los años de experiencia. Si el modelo cumple homocedasticidad, el "error" entre lo que predecimos y lo que realmente gana cada persona debería ser más o menos igual tanto para los que tienen 1 año de experiencia como para los que tienen 20. Además la varianza de los errores cambia con el valor de la variable y esto hace que los intervalos de confianza y pruebas estadísticas no sean confiables, porque se subestima o sobreestima la precisión de las estimaciones.

5. Normalidad del error

Dentro del contexto de machine learning en el análisis de regresión lineal, uno de los supuestos más importantes es que los errores (también llamados residuos) se distribuyen de forma aproximadamente normal. Esto significa que, si representamos gráficamente los errores, deberían seguir una forma de campana, concentrándose la mayoría alrededor de cero y disminuyendo su frecuencia a medida que se alejan en valores positivos o negativos. La importancia de este supuesto radica en que permite aplicar pruebas estadísticas e intervalos de confianza de manera confiable. Cuando los errores son normales, los estimadores obtenidos en el modelo tienen mejores propiedades, y los resultados de las pruebas de significancia (como el valor p o los intervalos) son válidos. Si no se cumple la normalidad, las predicciones del modelo pueden seguir siendo útiles, pero la inferencia estadística (como verificar la significancia de los coeficientes) puede ser menos precisa. Por eso, normalmente se revisa este supuesto con pruebas estadísticas específicas o mediante gráficos como el histograma de los residuos o el gráfico Q-Q.

Conclusiones

La regresión lineal es un modelo estadístico y de Machine Learning fundamental: sencillo, interpretable y ampliamente utilizado para predicciones cuando la relación entre variables es aproximadamente lineal. Su claridad y aplicabilidad la convierten en un punto de partida esencial en análisis y aprendizaje automático, aunque tiene supuestos importantes que deben ser verificados. En el caso de nuestro ejemplo con frutas, la regresión lineal es útil para estimar precios en función de la producción y los costos de transporte, aunque se debe tener precaución si existen factores externos que alteren la relación lineal.
